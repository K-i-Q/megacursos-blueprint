# MegaCursos Blueprint

Projeto web para gestão escolar com foco em alunos, turmas, mensalidades e rotinas administrativas.

## Requisitos

- Node.js 18+
- npm (ou gerenciador compatível)

## Desenvolvimento

```sh
npm install
npm run dev
```

## Build e preview

```sh
npm run build
npm run preview
```

## Tecnologias

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
